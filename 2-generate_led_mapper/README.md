# 🔹 Project 2 — Generate LED Mapper

## 📘 Description
This project demonstrates the use of the **VHDL `generate` statement** to build repeated hardware structures automatically. Instead of writing multiple instances manually, `generate` creates a scalable LED mapping structure.

## ⚙️ Inputs & Outputs
**Inputs**
- A vector or set of signals to be mapped (depends on design variant)

**Outputs**
- A vector of LED signals generated via the `generate` loop

## 🔌 Pin Mapping
Defined inside the UCF file (LEDs mapped to board pins).

## 📂 Files
- `generate_led_mapper.vhd`  
- `generate_led_mapper.ucf`  
- `generate_led_mapper.xise`  

## 🧠 Concepts Demonstrated
- VHDL `generate`  
- Scalable hardware replication  
- Clean multi‑bit signal mapping
